# 🤖 J.A.R.V.I.S.

> *Dreaming wasn't enough. So I started building it.*

Hi, I'm Guna. 👋

This repository is where I'm building my version of J.A.R.V.I.S. while learning Python from scratch.

Instead of waiting until I "know enough," I decided to build my dream project while learning. This repository is that journey.

J.A.R.V.I.S. isn't just another chatbot. My goal is to create an AI assistant that can listen, understand, remember, and help with real tasks—one feature at a time.

Every commit here represents something new I've learned.

---

## 🚀 The Vision

One day, I want J.A.R.V.I.S. to be able to:

- 🎙️ Listen to voice commands
- 🗣️ Speak naturally
- 🧠 Remember conversations
- 💻 Control my computer
- 🌐 Search the web
- 👀 Understand what's on my screen
- 🤖 Learn and improve over time

It's a long journey, but every project starts with a single line of code.
